Projects

E-Book Library

The E-Book Library project is a web-based digital library platform developed as part of the Bachelor of Computer Applications curriculum at D. Y. Patil International University, Akurdi, Pune. Built using Python’s Flask framework and integrated with Amazon Web Services (AWS), the project aims to provide a user-friendly, scalable, and secure solution for distributing electronic books (eBooks). This system supports key functionalities such as user registration, login, eBook browsing and downloading, feedback submission, and admin-level content management. The use of cloud technologies ensures high performance and scalability while reducing infrastructure complexity.
